# UniTV Recargas VIP - Plugin WordPress

Plugin WordPress para página de recargas UniTV.

## Instalação

1. Baixe o repositório como ZIP
2. No WordPress: Plugins → Adicionar Novo → Enviar Plugin
3. Selecione o ZIP e instale
4. Ative o plugin

## Shortcodes

- `[unitv_recargas]` - Página principal de recargas
- `[unitv_privacidade]` - Política de Privacidade
- `[unitv_termos]` - Termos de Uso
- `[unitv_reembolso]` - Política de Reembolso
- `[unitv_cookies]` - Política de Cookies
